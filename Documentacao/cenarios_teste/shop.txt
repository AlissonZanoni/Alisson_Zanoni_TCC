# Cenários de Teste - Loja

1. Validar o header da página
   Passos:
   1. Abra o navegador e insira a url "http://practice.automationtesting.in/".
   2. Clique no menu Shop.
   3. Visualize o header contendo Home/Shop
   4. O usuário pode voltar pra Home clicando no link

2. Validar as categorias dos produtos
   Passos:
   1. Abra o navegador e insira a url "http://practice.automationtesting.in/".
   2. Clique no menu Shop.
   3. Visualize as categorias dos produtos
   4. Cada categoria deve filtrar e exibir a quantidade de produtos corretamente

3. Validar filtro por Popularidade
   Passos:
   1. Abra o navegador e insira a url "http://practice.automationtesting.in/".
   2. Clique no menu Shop.
   3. Selecionar o filtro por Popularidade
   4. Deve reoordenar a lista dos produtos

4. Validar filtro por Avaliações
   Passos:
   1. Abra o navegador e insira a url "http://practice.automationtesting.in/".
   2. Clique no menu Shop.
   3. Selecionar o filtro por Avaliações
   4. Deve reoordenar a lista dos produtos

5. Validar filtro por Novidades
   Passos:
   1. Abra o navegador e insira a url "http://practice.automationtesting.in/".
   2. Clique no menu Shop.
   3. Selecionar o filtro por Novidades
   4. Deve reoordenar a lista dos produtos

6. Validar filtro por Preço Crescente
   Passos:
   1. Abra o navegador e insira a url "http://practice.automationtesting.in/".
   2. Clique no menu Shop.
   3. Selecionar o filtro por Preço Crescente
   4. Deve reoordenar a lista dos produtos

7. Validar filtro por Preço Decrescente
   Passos:
   1. Abra o navegador e insira a url "http://practice.automationtesting.in/".
   2. Clique no menu Shop.
   3. Selecionar o filtro por Preço Decrescente
   4. Deve reoordenar a lista dos produtos

8. Filtrar por Alcance de Preço
   Passos:
   1. Abra o navegador e insira a url "http://practice.automationtesting.in/".
   2. Clique no menu Shop.
   3. Ajuste o filtro de preço entre 100 e 200 rps.
   4. Clique no botão Filtrar.
   5. O usuário pode visualizar apenas livros com preço entre 100 e 200 rps.

9. Promoção
   Passos:
   1. Abra o navegador e insira a url "http://practice.automationtesting.in/".
   2. Clique no menu Shop.
   3. Tenha um produto em promoçao na listagem.
   4. Verifique o preço atual do produto em promoção.
   5. O usuário pode visualizar o preço atual com o preço antigo riscado para os produtos em promoção.

10. Exibição dos produtos
   Passos:
   1. Abra o navegador e insira a url "http://practice.automationtesting.in/".
   2. Clique no menu Shop.
   3. Deve ser apresentado a listagem dos produtos.
   4. Cada produto deverá conter título
   5. Cada produto deverá conter Preço.
   6. Cada produto deverá conter o botão para adicionar ao carrinho

11. Redirecionamento para o produto
   Passos:
   1. Abra o navegador e insira a url "http://practice.automationtesting.in/".
   2. Clique no menu Shop.
   3. Clicar em um produto da lista.
   4. Deverá ser redirecionado para a pagina do produto.
